# tic-tac-toe-game-cpp
A Tic-Tac-Toe game written in C++.

## Features

- Player vs AI gameplay
- Turn-based game logic
- Win, lose, and draw detection
- Input validation
- Console-based interface

## Technologies

- C++
- Functions
- Arrays
- Conditional statements
- Loops
  
## Goal

The goal of this project was to practice programming fundamentals, game logic, and decision-making algorithms in C++.
